Utilizando degradê com CSS

21/08/22

Como utlizar

Para fazer uso do degradê é bem simples, basta usar, dentro da CSS, a tag <background-image: linear gradient();

Dentro da tag é possível ajustar para qual direção o gradiente vai ficar, para a direita, para cima, esquerda, se vai ser por meio de valores em graus, etc. Além disso, é possível utilizar várias combinações de cores, as cores são colocadas logo após a direção do degradê.

comandos:
  • direita: to right
  • esquerda: to left
  • cima: to top
    • OBS:
    • quando for utilizar o degradê para cima é necessário usar, dentro da CSS, uma configuração global, dentro dela, utiliza-se a tag <height> definida para 100%.
  • baixo: to bottom
  • quando for usar o degradê usando valores em graus, a forma de escrita é: numeração + deg.
    • exemplo:
    • 90deg

Se o degradê não estiver cobrindo toda a página é necessário usar na CSS a tag <background-attachment> definida para fixed.
