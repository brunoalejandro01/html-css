Utilizando CSS para personalizar meu site

09/08/2022

Nessa aula foi ensinado sobre como utilizar o CSS para personalizar o meu site, utilizando cores, mudando fontes e etc. Para fazer uso das CSS é bem simples, elas são feitas na <head> e por meio da tag <style>, dentro dessa tag vamos utilizar os comandos para personalizar.

Como exemplo, podemos citar o seguinte caso, se eu quisesse mudar a cor e a fonte de tudo que está dentro do <body> eu iria realizar o seguinte procedimento, dentro da tag <style> eu colocaria a palavra body e abriria chaves, dentro dessas chaves eu iria colocar as persanalizações que eu quero que tudo que está dentro do <body> tenha. Um outro caso, se eu quiser mudar a cor da letra de todos os <h1>, basta eu colocar h1 dentro da tag <style> e abrir chaves, dentro das chaves eu colocaria as mudanças que eu gostaria de fazer e assim por diante.

OBS: esse estilo de CSS deve ser usado em casos onde só temos uma página de arquivo e para pequenos projetos.
OBS2: sempre observar se os atributos possuem ponto e vírgula no final.

Abaixo será listado algumas atributos que podem ser usados dentro da tag <style>

  • background-color
  • font-family
  • font-size
  • color
  • text-align
