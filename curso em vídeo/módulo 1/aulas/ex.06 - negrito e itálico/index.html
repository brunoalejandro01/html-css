Principais formatações de textos

02/08/2022

Nessa aula, aprendi sobre tipos de formatação, como por exemplo, negrito e itálico. Tais formatações servem para dar destaque ou ênfase para certas palavras/expressões contidas em um texto.

Cabe ressaltar que existem dois tipos de formatação, tanto para itálico como para negrito, sendo uma não semântica e outra semântica. Na qual, a versão não semântica apenas representa uma forma, ou seja, é apenas uma palavra colocada em negrito ou em itálico. Já a versão semântica quer representar algo que merece destaque, uma palavra forte, no uso do negrito, por exemplo.

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag <b>, versão não semântica.

Nesta frase, temos um termo em destaque usando a tag <strong>, versão semântica.

Itálico/Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>, versão não semântica.

Nesta frase, temos um termo em ênfase usando a tag <em>, versão semântica.

Atalho

Usando o atalho Ctrl + Shift + P, o Visual Studio vai abrir uma aba de pesquisa, nessa aba podemos colocar a opção "wrap with abbreviation" numa palavra ou frase selecionada e dizer qual tag queremos para aquela frase ou texto, caso seja necessário trocar a tag sem precisar apagar tudo, por exemplo.

Texto marcado

podemos inserir também marcações em uma palavra ou frase, como se tivéssemos grifado. Essa marcação é inserida usando a tag <mark>
exemplo: palavra

Texto grande e texto pequeno

é possível aumentar ou diminuir o tamanho de algumas palavras do texto usando tags, a tag para aumentar o tamanho da palavra é <big> e para diminuir a palavra usamos <small>. Contudo, a tag <big> é uma tag que entrará em desuso, logo, seu uso não é recomendado, visto que sua função é mais voltada para as CSS.
exemplo: grande
exemplo: pequeno

Texto deletado

podemos marcar um texto ou palavra com um risco sob a mesma, representado aquela palavra como uma palavra que pode ser lida, porém deve ser desconsiderada. Para isso, usa-se a tag <del>.
exemplo: catapimbas

Texto inserido

Um texto inserido serve para dar ênfase a alguma palavra ou frase, a palavra que recebe essa tag vem acompanhada de um risco embaixo dela e serve também para indicar que aquela palavra foi adicionada depois. Para usar essa formatação, usa-se a tag <ins>.
exemplo: teste
OBS: existe também a tag <u> que possui a mesma função, porém é uma versão não semântica.

Texto sobrescrito e subscrito

Um texto sobrescrito serve para , por exemplo, números com potência. Usa-se a tag <sup>.
exemplo: x20+10
Um texto subscrito serve para números ou letras que precisam ficar um pouco abaixo da linha do texto. Usa-se a tag <sub>.
exemplo: H2O
