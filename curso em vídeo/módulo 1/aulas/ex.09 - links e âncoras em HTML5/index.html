Usando links externos nos meus sites

é possível, obviamente, usar links internos dentro dos sites que eu faço, tais links levaram o usuário para outros sites já feitos. Para isso, usa-se a tag <a>, que vai possibilitar colocar um link no seu site, tal link pode ser atrelado a uma palavra, que vai agora possuir uma função além da textual, vai também ser um hyperlink.

OBS: é recomendado que ao usar links internos, tais links abram seus respectivos sites em uma outra aba, e não que abram por site do meu site. O que evita que o usuário não acesse mais meu site.
Para conseguir realizar tal função, além do link do site é necessário usar as palavras reservadas target, juntamente com a opção _blank e a outra palavra reservada rel, juntamente com a opção external.

exemplo: você pode acessar o site do sócio fortaleza aqui!

OBS:

  • Target: o target é um atributo que vai indicar ao navegador onde o site de destino vai abrir
    • target _blank: indica ao navegador que o link deve ser aberto em outra janela.
    • target _self: indica ao navegador que o link deve ser aberto na janela ou frame atual, é recomendado quando o link vai levar o usuário a outras páginas dentro do meu site.
  • Rel: o rel é outro atributo que vai indicar ao navegador qual o destino do link que foi colocado
    • rel next: indica ao navegador que o link é para próxima parte do documento atual
    • rel prev: indica ao navegador que o link é para a parte anterior do documento atual
    • rel external: indica ao navegador que o link é para ser aberto em outro site e não no site atual
    • rel unfollow: é um tipo de rel que vai indicar ao navegador que o link que foi colocado não recebe o aval do criador do site, esse tipo de rel é indicado quando o link que vai ser colocado não é muito confiável ou quando é um link para fazer propaganda, um link pago para está ali naquele site.

    Usando links internos em meu site

    Esta é a primeira página do meu site, para acessar a segunda página do meu site você pode clicar aqui!

    Para criar uma segunda página no próprio site, basta criar um novo arquivo, também html, dentro do arquivo principal. Após isso, o VSC vai criar uma segunda aba HTML. Agora, para que o usuário consiga acessar a próxima página, basta usar a tag <a> vinculado a uma palavra para transformá-la em hyperlink. Feito isso, víncule a palavra escolhida para servir como hyperlink o arquivo referente a segunda página do meu site por meio do atalho Ctrl + barra de espaço, por fim, lembrar de colocar as palavras reservadas target e rel.

    OBS: lembrar de usar o atalho Ctrl + barra de espaço para escolher o arquivo da segunda página dentro da tag <a>.

    O processo anterior também pode ser feito na segunda página para que o usuário consiga voltar para a página principal, sendo necessário somente a criação de um hyperlink vinculado a uma palavra escolhida, feito isso, basta usar o atalho Ctrl + barra de espaço e escolher o arquivo referente a primeira página.

    Criando uma página em pasta

    É possível também criar novas páginas no meu site por meio de diferentes pastas. Ou seja, dentro da pasta principal, onde encontra-se o arquivo html do meu site, eu posso criar uma nova pasta. Feito isso, na aba explorador do VSC, vai estar a nova pasta que foi criada, nela, deve-se criar um novo arquivo, também html.

    Para ir da página principal ou de qualquer outra página para essa nova página feita em uma pasta é necessário usar, também, o atalho Ctrl + barra de espaço. Esse comando deve ser usado tanto para ir como para voltar dessa nova página.

    Essa é minha página de notícias, você pode acessar ela aqui!

    OBS: quando uma nova página é feita em uma pasta diferente, pra voltar dessa página para a página principal, por exemplo, o comando é diferente, não da mais para usar o atalho Ctrl + barra de espaço. Portanto, vamos usar um novo comando, que é o dois pontos + barra, ../, esse comando serve quando queremos transitar entre diferentes pastas e é usado nesse tipo de situação.
