Código fonte ou pré-formatação

Nessa formatação, é possível adaptar o texto escrito para colocá-lo em formato de código. Usa-se a tag <code>.
exemplo: document.getElementById("teste").

O comando <code> é escrito em linguagem JS. Contudo, nem sempre ao escrever uma linha de código no VSC o código sairá formatado, se isso acontecer, é possível usar a tag <pre> junto com a <code>

Citações simples

Para demonstrar citações é possível usar a tag <q>. Também é possível simplesmente colocar as aspas normalmente, porém para citações é recomendado o uso da tag.
exemplo: é jogo de futebol vojvoda não é levantamento de peso não

Citações completas

Nesse caso, pode-se inserir citações em bloco, pegar um texto de algum site, por exemplo e colocar esse texto dentro da tag <blockquote>. Além disso, pode-se usar a palavra reservada "cite", que serve para colocarmos o link ou a fonte da citação para que o navegador relacione a citação com sua fonte, sendo mais fácil para encontrá-la.

Abreviações

Nessa formatação, é possível criar pop-ups nas abreviações para que, ao usuário passar o mouse sobre a abreviação, por exemplo, o significado da palavra apareça na tela. Usa-se a tag <abbr>.
