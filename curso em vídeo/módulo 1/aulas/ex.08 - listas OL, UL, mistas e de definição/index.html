Criação de listas em HTML

02/08/2022

Listas ordenadas - OL

As listas ordenadas são listas em que a ordem dos objetos presentes na lista importa.
Para criar uma lista ordenada basta usar a tag <ol>, além disso, dentro do comando <ol>, é necessário usar a tag <li>, que vai auxiliar na numeração dos itens da lista, mudando automaticamente os números ou letras em caso de adição ou remoção de um item da lista.

exemplo:

  1. Acordar
  2. abrir a janela
  3. beber água
  4. banheiro
  5. merendar
  6. tomar uma
Além disso, dentro da lista, podemos mudar o seu tipo de organização, mudando uma lista numerada para uma lista organizada por letras ou algarismo romanos. Para isso, fazemos uso da palavra reservada type dentro do comando <ol>. No mais, também podemos começar a partir de um número diferente de 1, usando a palavra reservada start dentro da tag <ol>.

Listas não ordenadas - UL

Nesse tipo de lista a ordem não importa, ela serve para fazer demarcações, ou somente para listar itens, por exemplo. Para utilizar esse tipo de lista, basta usar a tag <ul> acompanhada da tag <li>.

exemplo:

  • vermelho
  • azul
  • verde
  • preto

Esse tipo de lista também pode ser personalizada por meio do comando "type", dando as opções: disc, circle, square.

Listas mistas

Listas mistas são listas em que ocorre a junção de listas do tipo <ol> e <ul>, na qual dentro de uma <ol> podemos ter uma <ul> e vice versa.

Aqui abaixo contém um exemplo de lista mista usando <ol> dentro de outros <ol>
exemplo: meus jogos favoritos

  1. Antigos
    1. GTA - San Andreas
    2. Black
    3. Brasfoot
    4. Club penguin
    5. Mundo gaturro
  2. Novos
    1. The witcher
    2. GTA 5
    3. BF4
    4. Far cry 3

Aqui abaixo conterá uma lista mista usando <ul> dentro de <ol>
exemplo: meus cantores/as favoritos/as

  1. Taylor Swift
    • Exile
      • minha preferida
    • All to well
    • Getaway car
  2. Harry Styles
    • Falling
    • Matilda
    • Sweet creature

Lista de definição

Esse tipo de lista funciona como se fosse um dicionário. Ela usa as tags <dl>, de definition list, e dentro da <dl> tem outras duas tags, a <dt>, que serve para o termo, e a <dd>, que serve para descrição do termo.

exemplo:

GTA
Grand Theft Auto.

Atalho

Podemos colocar o curso do mouse entre as letras de uma tag e, segurando a tecla Alt, podemos ir selecionando varias tags para apagar uma letra específica da palavra, no caso de querer mudar, por exemplo, de <ol> para <ul>.
