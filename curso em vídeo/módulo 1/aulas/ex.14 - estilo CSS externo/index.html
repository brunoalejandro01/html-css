Utlizando CSS externo nos meus sites

10/08/2022

Como utilizar

O CSS externo é mais um tipo de CSS, porém, diferentes dos anteriores, ele não está junto com o código HTML em si, pois, como o próprio nome diz, ele é externo, se localiza em um segundo arquivo.

Portanto, para utilizá-lo, basta fazer o seguinte, usa-se a tag <link> dentro da <head>, na tag, vamos escolher a opção link:css, ao colocar a tag, ela vai possuir dois atributos, no segundo, vai ficar o nome do arquivo CSS, que pode ser modificado para qualquer outro nome, e é por meio dele que vamos fazer o arquivo CSS. Para criar o arquivo é bem simples, basta clicar no nome definido para a CSS com o botão Ctrl pressionado, o VSC vai reclamar que não existe nenhum arquivo com esse nome e vai dar a opção de criar um nova, ao selecionar para criar um arquivo CSS, uma nova aba vai ser aberta e é nessa aba que o CSS vai ser feito.

Nessa nova aba ficarão todas as modificações que serão feitas no site, ela é uma aba geral e pode ser aplicada em todas as páginas do projeto, basta colocar, em cada uma das páginas o link:css e o nome definido para o arquivo CSS. Como ela é uma aba geral, qualquer modificação feita nela vai ser aplicada a todas as páginas do projeto que contém o link:css.

Além disso, pode ser feito mais de um <link>, assim, podem ser criadas modificações específicas para certas páginas do projeto, bastando mudar o nome do arquivo CSS. Por fim, as outras CSS também podem ser aplicadas no projeto, como a interna e a inline, pois como são modificações para páginas ou tags específicas, elas não vão afetar o projeto como um todo.

A CSS externa é a mais recomendada, sobretudo em projetos com mais de uma página.
