Colocando imagens adaptativas no meu site

08/08/2022

Nessa aula foi ensinado sobre adaptação de imagens para diferentes tamanhos de tela. Uma mesma imagem deve ser adaptada para diferentes tipos de tela, logo é necessário mudar seus tamanhos para que ela se adapte automaticamente. Para isso, vamos usar uma nova tag, a <picture>, dentro dela, vamos usar a tag <img> para colocar as imagens que vamos usar.

OBS: dentro da tag <img> vamos usar uma outra tag, a <source:media type>, é nessa tag que vamos conseguir adaptar as imagens para diferentes tamanhos de tela. Dentro da tag <source< vamos colocar a opção de largura no máximo e sempre deixando uma margem de 50 a mais em relação ao tamanho da imagem. Além disso, dentro da opção srcset coloca-se o nome da pasta e o nome da imagem utilizada e no type coloca-se o tipo da imagem, png, por exemplo.

Dentro dos source a lógica é a seguinte, começamos colocando a imagem grande, e supondo que ela seja de tamanho 1000, quando formos colocar o source para a imagem média vamos pensar assim, se o navegador tiver um tamanho de tela menor que 1050 (imagem grande + barra de rolagem), use a imagem média. A mesma lógica serve para o próximo source, se o tamanho da tela for menor que 700 (imagem média + barra de rolagem), use a imagem pequena.

OBS: para cada novo <source:media:type> colocado, deve-se colocar esse novo em cima do outros que já estavam no código, nunca colocar depois, e lembrar sempre de seguir uma lógica, começou colocando a imagem grande, então a próxima é a média e por último a pequena.

Colocando áudios no meu site

Para conseguir colocar áudios em um site é bastante simples, basta usarmos uma nova tag, a <audio> e, junto com ela, algumas <sources> que vão ajudar com as configurações do áudio.

Alguns atributos usados dentro da tag <audio>

  • preload: indica se o áudio será pré-carregado ou não e aceita três parâmetros
    • metadata: vai carregar apenas as informações sobre o arquivo de áudio, como seu tamanho, a duração, direitos, etc.
    • none: não vai carregar nada até o usuário clicar no botão de play do áudio ou algum script iniciar a reprodução.
    • auto(padrão): vai carregar o arquivo de áudio automaticamente assim que a página for carregada, independente do usuário ter apertado o play ou não.
  • controls: vai apresentar a central de controle na tela do site.
  • autoplay: vai iniciar a reprodução do áudio assim que a página for carregada.
  • loop: vai fazer com que o áudio seja reproduzido infinitamente.

OBS: evitar usar arquivos de áudio do tipo wav, pois tais arquivos são grandes e podem deixar o site lento.
