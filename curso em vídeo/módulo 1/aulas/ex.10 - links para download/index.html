Como criar links para download no meu site

04/08/2022

Para colocar um link com arquivo para download em um site é bem simples, basta usar a tag <a>. Nela vamos colocar o arquivos, ou os arquivos, que vamos disponibilizar aos usuários para serem baixados.
OBS: certos navegadores ao invés de baixarem o arquivo, vão somente abrí-lo, nesse caso, podemos usar dois comandos, o primeiro é o download, que vai receber o nome do arquivo que foi colocado, e o segundo comando é o type, que vai receber parâmetros como application/pdf, para arquivos PDF, ou o application/zip, que recebe arquivos ZIP. Tais comandos devem estar juntos com a tag <a>.
