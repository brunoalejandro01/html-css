Tags em HTML e CSS

10/08/2022

    Títulos
  • Usa-se a tag <h1> até a <h6>, sendo a <h1> a de maior importância, utilizada para o título principal, por exemplo, e a <h6> a de menor importância.

    • Parágrafos
  • Usa-se a tag <p>.

    • Espaço entre as linhas ou quebra de linhas
  • Usa-se a tag <br>

    • Adicionar uma linha horizontal
  • Usa-se a tag <h>

    • Inserindo emojis
  • Para inserir emojis basta ir em algum site que disponibilize emojis para emojis para baixar, como o emojipedia, nele, basta escolher um emoji e clicar no seu codepoints, com o código do emoji em mãos, exemplo: U+1F604, basta substituir esse U+ por &#x no código.

    • Inserindo imagens
  • Para inserir uma imagem basta usar a tag <img>, nela serão criado dois parâmetros, o primeiro será o src que é onde vamos inserir o arquivo de imagem, e o segundo será o alt, que é onde coloca-se um texto alternativo, no caso da imagem não carregar, que é de suma importância por conta da acessibilidade, então é importante fazer uma boa descrição da imagem.
    • é possível também inserir imagens que estão em pastas diferentes, contanto que essa pasta esteja dentro da pasta onde está o arquivo html principal. Nesse caso, para ter acesso a essa segunda pasta, basta usar o atalho Ctrl + barra de espaço dentro do src e escolher a pasta que contém a imagem.
    • além disso, também é possível pegar o link da imagem e inserir diretamente na tag.

    OBS: sempre se atentar para dois fatores quando for utilizar imagens em meu site
    • a maioria das imagens possuem direitos autorais, portanto, sempre optar por imagens de autoria própria ou do cliente, ou usar banco de imagens que não possuam direitos autorais, como em sites como o pexels ou o UnSplash.
    • outro ponto importante é estar atento ao formato da imagem e seu tamanho. No caso do formato, sempre optar pelo JPEG, com uma compactação entre 30% e 50%. O formato PNG é recomendado em casos no qual a imagem precisa de transparência.
    • lembrar de adaptar o tamanho da imagem por meio de aplicativos de edição, como o gimp.

    Inserindo favicons
  • Favicons são aquele pequeno símbolo que fica na aba do site, para colocar em meu site basta usar a tag <link> e escolher a opção link:favicon, dentro dessa tag vão existir três parâmetros, no primeiro não se mexe, já no segundo é onde colocaremos nosso favicon, ele já vem com um por padrão, portanto basta apagar e colocar o meu favicon, no último também não se mexe.
    • o principal formato para o favicon é o .ico, mas o PNG também é aceito.
    • site para achar um favicon: iconArchive
    • site para desenhar seu próprio favicon: favicon.cc
    • site para criar um favicon personalizado: favicon.io

    Criar um lorem
  • Basta escrever <lorem>.

    • Deixar a palavra em negrito
  • Basta usar a tag <stron>.

    • Deixar a palavra em italico
  • Basta usar a tag <em>.

    • Deixar as letras pequenas
  • Usa-se a tag <small>.

    • Texto deletado
  • Usa-se a tag <del>, esse tipo de comando é recomendado quando o texto pode até ser lido, mas deve ser desconsiderado.

    • Texto inserido
  • Usa-se a tag <ins>, nesse caso, esse comando é recomendado quando é necessário dar mais atenção ao texto ali marcado.

    • Texto sobrescrito e subscrito
  • Para o texto sobrescrito usa-se a tag <sup> e para o texto subscrito usa-se a tag <sub>.

    • Trechos de código
  • Quando formos inserir linhas de código no HTML é necessário usar as tags <pre> e <code> juntas, assim o código vai ficar pré-formatado e de fácil leitura.

    • Citações
  • Usa-se a tag <q>. Para citações em bloco usa-se a tag <blockquote>. A diferença entre elas é que a primeira não ganha recuo de texto, já na segunda existe o recuo. Ainda é possível usar a tag <cite> dentro da <blockquote> para colocar o link da citação.

    • Abreviações ou pop-ups
  • Usa-se a tag <abbr>, dentro dela, no parâmetro title colocamos o significado da abreviação que colocamos.

    • Listas
  • Para listas ordenadas, ou seja, onde a ordem importa, usa-se a tag <ol>.
  • Para listas onde a ordem não importa, usa-se a tag <ul>.
  • é possível personalizar o tipo de lista usando o atributo type dentro do comando da lista, sendo possível criar listas numéricas, alfabéticas ou em números romanos.
  • Também é possível criar listas de definição por meio da tag <dl>, dentro dessa tag existem dois parâmetros
    • <dt>: é o termo de definição, ou seja, o termo que vai ser definido.
    • <dd>: é a descrição do termo.
